# prompt

Custom prompt functions

## Installation (zsh)

```bash
git clone https://mwinokan/prompt && cd prompt
echo source $PWD/zsh.sh >> .zshrc

# optional splashscreen
echo bash $PWD/splash.sh >> .zshrc
```

## Installation (bash)

```bash
git clone https://mwinokan/prompt && cd prompt
echo source $PWD/bash.sh >> .bashrc

# optional splashscreen
echo bash $PWD/splash.sh >> .bashrc
```

## Installation (hpc bash)

```bash
git clone https://mwinokan/prompt && cd prompt
echo source $PWD/hpc.sh >> .bashrc

# optional splashscreen
echo bash $PWD/splash.sh >> .bashrc
```
